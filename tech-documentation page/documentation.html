What is Python ?

Python is programming language. Python is high level easy to type , easy to learn language. Python is not strictly typed and python code is generally smaller in terms of lines.

Features of Python:

  • High-level programming language: With Python, the code looks very close to how humans think. For this purpose, it must abstract the details of the computer from you: memory management, pointers,… Hence, it is slower than “lower-level language” like C;

  • Python is interpreted and not compiled: Sure, this statement is a gross simplification but it’s somehow correct. During the execution, Python code is interpreted at runtime instead of being compiled to native code at compile time;

  • Python is a dynamically typed language: Unlike “statically-typed” languages like C, C++ or Java, you don’t have to declare the variable type like String, boolean or int. The less you do, the more your computer has to work. For each attribute access, tons of lookup is required. In addition, being very dynamic makes it incredibly hard to optimize Python;

  • Global Interpreter Lock (GIL): This GIL basically prevents multi-threading by mandating the interpreter only execute a single thread within a single process (an instance of the Python interpreter) at a time.

  • More Productive:First and foremost reason why Python is much popular because it is highly productive as compared to other programming languages like C++ and Java. It is much more concise and expressive language and requires less time, effort, and lines of code to perform the same operations.

If else

In programming we need to use conditional jumps for program execution. We can use if-else block for that. We can also use if-elif-else or nested if-else too!

If else statement in python :

#if else
if name=='python': 
   print('Its python!')
else: 
   print('Its not python!)


#if-elif-else
if i=='a': 
   print('Its a')
elif i=='b': 
   print('Its b')
else: 
   print('anything but not a and b')


#nested if-else
if number>10: 
   if number>5
       print('Number is smaller than 5')
   else:
       print('Number is greater that 5')
else: 
   print('Number is greater than 10!')
For loop

When we need to perform a task over and over we use for loop. We can specify how many times block of code should run.

for loop in python:

# for loop that prints 1 to 10
for i in range(1,10): 
   print(i)
